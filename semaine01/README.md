# Semaine 1 – Révision bases HTML/CSS/JS

Objectifs
- Réviser la structure HTML5 sémantique (header/nav/main/footer, section/article)
- Réviser flexbox, sélecteurs, transitions/transformations CSS
- Manipulation minimale du DOM (`querySelector`, événements de souris)

Exercices suggérés
1) Ajouter une section "Équipe" avec 3 cartes.
2) Ajouter un léger effet `:hover` sur les cartes (mise à l'échelle/ombre).
3) Brancher un clic sur les tuiles des films et afficher un message dans la console.
