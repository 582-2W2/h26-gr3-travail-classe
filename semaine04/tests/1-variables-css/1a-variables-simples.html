Propriétés personnalisées CSS (Variables CSS) - À COMPLÉTER

À FAIRE: Déclarer les variables CSS dans :root avec --nom-variable: valeur;
Puis les utiliser partout avec var(--nom-variable)

1. Boutons avec couleurs centralisées

À FAIRE: Si on change --couleur-primaire en :root, tous les boutons primaires se mettent à jour!

2. Cartes avec espacement variable

À FAIRE: Ajouter display: grid et utiliser grid-template-columns avec repeat(auto-fit, minmax(250px, 1fr))

Titre 1

Le padding utilise var(--espace-grand). L'espacement est cohérent dans toute l'interface.

Titre 2

Tous les radius utilisent var(--radius). Modifier :root change tous les arrondis.

Titre 3

Les couleurs aussi! var(--couleur-primaire) s'applique aux titres et boutons.

3. Étiquettes de statut

À FAIRE: Ajouter display: flex
